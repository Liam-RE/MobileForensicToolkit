
# Mobile Device Forensic Toolkit

A tool developed in Python that automates the process of extracting and parsing data from Android and iOS devices using a range of open-source tools! 

# Installation / Getting Started
## Debian / Ubuntu Linux  

To install the toolkit run.

```bash
  ./install.sh
```

Once installed run.


```bash
python3 main.py 
```


## Features

- Logical, Filesystem and Physical on certain devices.
- i/ALEAPP and Autopsy forensic tools.
- SFTP Data Extraction.
- .E01 conversion of image files
- Mirror and control Android devices.
- Image extractor for iOS devices.
- Key file search.
- WhatsApp message parsing (Android).
